# 나이아델 RPG 시뮬레이터 v5

HTML, CSS, JavaScript만으로 구성한 나이아델 간단 RPG 전투 시뮬레이터다.

## v5 수정 사항

- 캐릭터 방향 시스템 추가
  - 이동 목표가 현재 위치보다 오른쪽이면 우측 방향 애니메이션 사용
  - 이동 목표가 현재 위치보다 왼쪽이면 좌측 방향 애니메이션 사용
  - 몬스터를 공격하거나 스킬을 사용할 때 대상의 위치를 기준으로 자동 방향 전환
- 우측 기준 캐릭터 스프라이트 추가
  - 기존 좌측 기준 프레임을 기준으로 우측 방향 프레임을 별도 PNG로 생성
  - `idle_right_0~5.png`
  - `move_right_0~7.png`
  - `attack_right_0~5.png`
  - `skill_right_0~7.png`
- 기존 v4 기능 유지
  - 캐릭터 애니메이션 중앙 정렬 보정
  - 기본 공격, Q/W 스킬 0.8초 준비 시간
  - E 스킬 차지 게이지 및 1.2초 차지 발동
  - 몬스터 좌클릭 기본 공격
  - 대상 정보 / 전투 로그 / 캐릭터 상세 스크롤
  - 고정 UI 레이아웃

## 조작

- 우클릭: 이동
- 몬스터 좌클릭: 대상 선택 및 기본 공격
- Q: 방패 가격
- W: 전방 베기
- E 길게 누르기: 돌진 찌르기 차지

## 파일 구성

- `index.html`
- `style.css`
- `game.js`
- `assets/knight/` 캐릭터 애니메이션 프레임


## v6 변경 사항

- 최종 피해량에 -5% ~ +5% 난수 보정을 적용했다.
- 적용 위치: `damageFormula()`
- 적용 대상: 플레이어 기본 공격, Q/W/E 스킬 피해, 몬스터 기본 공격 피해
- 계산 순서: 기본 피해 산출 → 난수 보정값 적용 → 소수점 버림 → 최소 피해 1 보정

```js
const variance = 0.95 + Math.random() * 0.10;
finalDamage = rawDamage * variance;
```
